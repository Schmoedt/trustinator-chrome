Plugin temporär einbinden:

1. Link in Adressleiste eingeben: chrome://extensions/ 
2. Auf Button "Entpackte Erweiterung laden" auswählen
3. Ganzes Verzeichnis auswählen

Falls der Link nicht funktioniert:
1. Einstellungen öffnen (Logo ganz rechts mit drei Punkten)
2. Weitere Tools -> Erweiterungen auswählen
3. Auf Button "Entpackte Erweiterung laden" auswählen
4. Ganzes Verzeichnis auswählen
